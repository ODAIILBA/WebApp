PROBLEM:
--------
The "Add to Cart" button was not working. Clicking it did nothing.

ROOT CAUSE:
-----------
The addToCart() function was trying to access 'event.target' but the 
'event' object was not in scope when called from onclick handlers.

This is because onclick="addToCart(1, 1)" doesn't automatically pass 
the event object - it must be explicitly passed.

SOLUTION:
---------
1. Updated function signature:
   async function addToCart(productId, quantity = 1, event = null)
                                                      ^^^^^^^^^^
                                                      Added this!

2. Updated button click logic to handle event:
   const target = event?.target || window.event?.target
   const button = target?.tagName === 'BUTTON' ? target : target?.closest('button')

3. Updated all onclick handlers to pass event:
   onclick="addToCart(1, 1, event)"
                            ^^^^^
                            Added this!
